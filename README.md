# 🔐 Random Password Generator

---

## 📌 Project Overview

This project is a **Random Password Generator Web Application** built using **HTML, CSS, and JavaScript**.
It allows users to generate secure, random passwords instantly and copy them with a single click.

The application focuses on:

* Clean UI design
* Real-time interactivity
* Practical use of JavaScript concepts

## 🧩 How It Works

1. User clicks on **"Generate Password"** button
2. JavaScript generates random characters
3. Characters are combined into a password string
4. Two passwords are displayed
5. User clicks on copy icon 📋
6. Password is copied to clipboard
7. "Copied!" message is displayed

---

## 🔐 Password Generation Logic

The password is generated by:

* Combining:

  * Alphabets (A-Z, a-z)
  * Numbers (0-9)
  * Symbols (!@#$...)

* Selecting random characters using:

```plaintext
Math.floor(Math.random() * length)
```

* Repeating the process in a loop
